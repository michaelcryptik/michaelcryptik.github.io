שם הפרוייקט

שם מגיש: מיכאל לפין

שם המנחה: גיא לשם

This is a research project that answers the question when should a heavy cost algorithm be implemented parallelly on the graphics card instead of running on a multi CPU server.

We answer this question by porting an existing physics program to a CUDA program that runs on graphcs cards and comparing its executing speed on a multi CPU server and on graphics cards so that we can deduce for what size problem a parallel implementation on GPU will be faster and worth to implement compared to running it on a powerful server.

Technology

CUDA - Parallel computing platform and API developed by Nvidia that allows software to use graphics cards for general purpose computing
